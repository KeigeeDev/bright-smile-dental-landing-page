# Bright Smile Dental Landing Page

This is a **static landing page** built with a **mobile-first approach**. 

> **Note:** This is **not a client project**. It is a concept project designed to practice **semantic HTML** and **responsive vanilla CSS**.

## Features
- Mobile-first design for optimal viewing on all devices
- Semantic HTML structure
- Responsive Vanilla CSS styling
- Zero external frameworks - pure web fundamentals

## Technologies Used

- CSS3
